# Gatekeeper.js

The JavaScript library for event delagation for mobile browser.

## Test enviroments

This library tests below browsers.

- Chrome 21.0.1180.89, OS X 10.7 (Lion)
- Safari 6.0, OS X 10.7 (Lion)

Maybe other browser can be used. But I don't test these browsers.

## Problem?

Please report [issues](https://github.com/Layzie/gatekeeper.js/issues).
